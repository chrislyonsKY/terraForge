# EarthForge Example Outputs

This directory contains output images and files generated by EarthForge example scripts. Every file uses **real-world data** from credible public sources — no simulated or synthetic outputs.

## Standards

### Accessibility (WCAG 2.1 AA)

- **Colorblind-safe palettes only** — viridis/cividis (sequential), brown-white-teal BrBG (diverging), ColorBrewer Set2/Paired (categorical). All palettes are defined in `earthforge.core.palettes`.
- **Text alternatives** — Every `.png` or `.gif` has a `.txt` sidecar file containing alt text suitable for screen readers.
- **Minimum text size** — 10pt at output resolution for all labels, titles, and annotations.

### Required Cartographic Elements (every map image)

1. **Title block** — dataset name, date, EarthForge version
2. **Colorbar** — labeled ticks with units
3. **CRS annotation** — e.g., "EPSG:32616 / UTM Zone 16N"
4. **Scale bar or coordinate labels**
5. **North arrow** (if not obviously north-up)
6. **Data source attribution**

### Data Citation

Each `.txt` sidecar includes:

```
Alt Text: [descriptive text for screen readers]

Data Source: [provider], [dataset name]
URL: [access URL]
Access Date: [YYYY-MM-DD]
License: [license identifier]
Spatial Extent: [bbox or region description]
Temporal Extent: [date range if applicable]

Generated By: earthforge v[version]
Script: examples/scripts/[script_name].py
Parameters: [key parameters used]
Generated: [YYYY-MM-DD HH:MM UTC]
```

### Audit Trail

Every output file is traceable to:
1. The script that generated it (in `examples/scripts/`)
2. The exact data source with access date
3. The EarthForge version used
4. The parameters passed to the generating command
